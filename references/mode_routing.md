# Mode Routing

## 默认策略

默认不要直接进入完整问卷。
先输出一版角色草案，再决定用户要哪种深度。

## Fast
- 触发条件：用户想尽快出结果 / 已有较多参考图 / 没想清楚细节
- 提问量：2-4 个关键问题
- 使用维度：仅关键维度
- 自动补全：最高

## Pro
- 触发条件：用户明确要求精细定义 / 长期 IP / 品牌虚拟人 / 高一致性
- 提问量：完整模块化提问
- 使用维度：完整 8 层
- 自动补全：最低

## 关键维度
- 角色定位 / 功能
- 年龄 / 性别表达 / 身份
- 整体美学方向
- 核心服装与造型方向
- 1-2 个记忆点

## 扩展维度
- 具体脸型
- 眼型 / 眼色
- 发丝与发质细节
- 手部与局部细节
- 局部材质层级
- continuity 规则
